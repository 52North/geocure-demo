Geocure REST API Demo

Using Leaflet, jQuery etc.

Click on the service you want to view:

    You will see a map on the right displaying the service's data.

    The control in the top right hand corner allows to choose between basemaps and to add map overlays from the WMS. The vector features from the WFS are loaded automatically.

    Note: Some of the feature sets are bigger than 1MB, and one is bigger than 20MB, so it might take a while until everything appears (the green box will take the longest).
